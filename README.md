## How to Run?
1. Create the `resources` folder.
2. Put the [image frames](https://github.com/trung-kieen/bad-apple-ascii/tree/main) in the folder (`out0001.jpg.txt` to `out6572.jpg.txt`)
3. Compile and run
```sh
./compile.sh
./run.sh
```
